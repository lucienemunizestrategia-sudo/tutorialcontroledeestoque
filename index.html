PASSO 1: Instalação do Sistema

IMPORTANTE - Antes de começar:

Você precisa ter uma conta do Gmail (Google). Se não tem, crie uma gratuitamente em gmail.com

1Baixar os arquivos

Clique no botão amarelo no topo desta página para baixar o arquivo .zip

Extraia (descompacte) os arquivos em uma pasta no seu computador

2Abrir o Google Apps Script

  1. Entre no seu Gmail
  2. Clique nos 9 pontinhos no canto superior direito
  3. Procure por "Apps Script"
  4. Se não encontrar, digite na barra do navegador: script.google.com
  5. Clique em "Novo projeto"

3Instalar o código principal

  1. Apague todo o código que já estiver na tela
  2. Abra o arquivo Codigo.gs que você baixou
  3. Copie todo o conteúdo (Ctrl+A, depois Ctrl+C)
  4. Cole no Apps Script (Ctrl+V)
  5. Clique em "Salvar" (Ctrl+S)
  6. Dê um nome para o projeto: "Sistema Estoque - [Seu Nome]"

4Adicionar a interface (HTML)

  1. Clique no ícone "+" ao lado de "Arquivos"
  2. Selecione "Arquivo HTML"
  3. Nome do arquivo: index
  4. Apague todo o conteúdo que aparecer
  5. Abra o arquivo index.html que você baixou
  6. Copie todo o conteúdo e cole no Apps Script
  7. Salve novamente

5Autorizar o sistema

  1. No código principal, encontre a função testarSistema
  2. Clique em "Executar"
  3. IMPORTANTE: Vai pedir permissões - clique em "Autorizar"
  4. Se aparecer "App não verificado", clique em "Avançado"
  5. Depois clique em "Ir para [nome do projeto]"
  6. Clique em "Permitir"

6Publicar seu sistema

  1. Clique em "Implantar" (canto superior direito)
  2. Selecione "Nova implantação"
  3. Clique no ícone de engrenagem e escolha "Aplicativo da Web"
  4. Configurações:
    • Executar como: Eu
    • Quem tem acesso: Qualquer pessoa com o link
  5. Clique em "Implantar"
  6. COPIE E GUARDE O LINK que aparecer - esse é o endereço do seu sistema!

DICA IMPORTANTE:

Salve o link do seu sistema nos favoritos do navegador ou anote em algum lugar seguro. Esse será o endereço que você usará para acessar seu sistema de estoque sempre.

PASSO 3: Cadastrando Seus Produtos

EXEMPLO PRÁTICO: Cadastrar "Arroz 5kg"

Vamos cadastrar seu primeiro produto passo a passo:

  1. Nome: Arroz 5kg Tio João
  2. Categoria: Alimentação
  3. Quantidade: 10 (quantos você tem agora na loja)
  4. Preço Compra: 4,50 (quanto você pagou por unidade)
  5. Preço Venda: 6,00 (quanto você cobra do cliente)
  6. Estoque Mínimo: 2 (quando avisar que está acabando)

Depois clique em "Adicionar Produto"

Nome do Produto

Seja específico:

  • ❌ "Arroz"
  • ✅ "Arroz 5kg Tio João"
  • ✅ "Detergente 500ml Ypê Neutro"
  • ✅ "Refrigerante 2L Coca-Cola"

Categorias Sugeridas

  • Alimentação: arroz, feijão, óleo, biscoito
  • Bebidas: refrigerante, água, cerveja, suco
  • Limpeza: detergente, sabão, desinfetante
  • Higiene: shampoo, sabonete, pasta de dente
  • Outros: pilhas, carregadores, etc.

Preços

Preço de Compra: Quanto você pagou no fornecedor

Preço de Venda: Quanto você cobra do cliente

Dica: Use ponto ou vírgula para separar os centavos (4.50 ou 4,50)

Estoque Mínimo

Quando chegar nessa quantidade, o sistema vai avisar que precisa comprar mais.

Sugestão:

  • Produtos que vendem rápido: 3-5 unidades
  • Produtos que vendem devagar: 1-2 unidades

PASSO 4: Registrando Movimentações

Toda vez que entra ou sai produto do seu estoque, você deve registrar. Isso mantém o controle sempre correto.

ENTRADA

Quando usar: Chegou mercadoria nova

Exemplo:

  • Produto: Arroz 5kg Tio João
  • Tipo: Entrada
  • Quantidade: 5 (chegaram mais 5 unidades)
  • Observações: "Compra da semana"

SAÍDA

Quando usar: Vendeu algum produto

Exemplo:

  • Produto: Refrigerante 2L Coca
  • Tipo: Saída
  • Quantidade: 3 (vendeu 3 unidades)
  • Observações: "Venda para dona Maria"

AJUSTE

Quando usar: Corrigir o estoque

Exemplo:

  • Produto: Detergente 500ml
  • Tipo: Ajuste
  • Quantidade: 8 (quantidade correta total)
  • Observações: "Acerto após inventário"

IMPORTANTE sobre Saídas:

O sistema não deixa você fazer uma saída que deixaria o estoque negativo. Se isso acontecer, verifique se a quantidade do produto está correta no sistema.

PASSO 6: Usando o Sistema no Dia a Dia

1Rotina da Manhã

  1. Abra o sistema no seu celular
  2. Verifique se tem produtos em baixa (aparecem em vermelho)
  3. Anote quais produtos precisa comprar
  4. Clique em "Atualizar" para ter os dados mais recentes

2Quando Chegar Mercadoria

  1. Conte os produtos que chegaram
  2. No sistema, vá em "Movimentação"
  3. Para cada produto, registre uma "Entrada"
  4. Coloque observações como "Compra do atacado" ou "Fornecedor João"

3A Cada Venda

Você pode escolher uma das opções:

  • Opção 1: Registrar na hora de cada venda (mais preciso)
  • Opção 2: Anotar as vendas e registrar no final do dia
  • Opção 3: Fazer um "inventário rápido" no final do dia e ajustar

4Final do Dia

  1. Registre todas as vendas do dia (se ainda não fez)
  2. Verifique se apareceram novos produtos em baixa
  3. Anote mentalmente quais produtos venderam mais

5Final do Mês

  1. Clique em "Fechamento do Mês"
  2. O sistema salva um "retrato" do seu estoque naquele momento
  3. Compare com o mês anterior para ver o crescimento

PASSO 8: Dicas para o Sucesso

SEMPRE FAÇA

  • Registre TODAS as entradas e saídas
  • Mantenha os preços sempre atualizados
  • Faça o fechamento mensal
  • Verifique produtos em baixa diariamente
  • Use nomes específicos para os produtos
  • Mantenha backup da planilha

EVITE FAZER

  • Mexer na aba "Configurações"
  • Apagar produtos que já tiveram vendas
  • Esquecer de registrar movimentações
  • Deixar preços zerados
  • Usar nomes genéricos ("refrigerante")
  • Ignorar alertas de estoque baixo

COMO DEFINIR PREÇOS

  • Preço de Compra: Exatamente o que você pagou
  • Preço de Venda: Inclua impostos + lucro
  • Margem sugerida: 30-50% sobre o preço de compra
  • Exemplo: Comprou por R$ 4,00 → Venda R$ 6,00

COMO DEFINIR ESTOQUE MÍNIMO

  • Produtos populares: 3-5 dias de venda
  • Produtos normais: 1-2 semanas
  • Produtos sazonais: Conforme a época
  • Exemplo: Vende 2 por dia → Mínimo = 6

DICA DE OURO para Crescer:

Use os relatórios mensais para identificar:

  • Quais produtos vendem mais (compre mais deles)
  • Quais produtos vendem menos (talvez não vale a pena)
  • Se o valor do seu estoque está crescendo mês a mês
  • Se você está tendo lucro real

PASSO 9: Resolução de Problemas

Sistema não carrega

Possíveis causas:

  • Internet lenta ou instável
  • Precisa autorizar novamente
  • Cache do navegador

Soluções:

  1. Verifique sua conexão
  2. Atualize a página (F5)
  3. Limpe o cache do navegador
  4. Tente em outro navegador

Números não batem

Possíveis causas:

  • Esqueceu de registrar algumas vendas
  • Registrou quantidade errada
  • Produto foi perdido/quebrado

Soluções:

  1. Conte fisicamente o produto
  2. Use "Ajuste" para corrigir
  3. Coloque observação explicando o ajuste

Não consegue fazer saída

Causa: Não tem produto suficiente no estoque

Soluções:

  1. Verifique se a quantidade no sistema está certa
  2. Se necessário, faça um ajuste primeiro
  3. Depois registre a saída

Perdeu acesso ao sistema

Soluções:

  1. Procure o link nos favoritos do navegador
  2. Entre no Google Drive e procure "Estoque_Prospera_Rio"
  3. Se necessário, refaça a implantação no Apps Script
  4. Entre em contato conosco

Se nada funcionar:

1. Respire fundo - seus dados estão seguros no Google Drive

2. Procure a planilha "Estoque_Prospera_Rio" no seu Drive

3. Anote os problemas que está enfrentando

4. Entre em contato conosco pelo WhatsApp ou email

PASSO 10: Backup e Segurança

Como Fazer Backup

  1. Clique em "Abrir Planilha"
  2. No Google Sheets, vá em Arquivo → Download
  3. Escolha "Microsoft Excel (.xlsx)"
  4. Salve em um local seguro (pendrive, nuvem, etc.)
  5. Faça isso pelo menos uma vez por mês

Segurança dos Dados

  • Seus dados ficam no Google Drive (muito seguro)
  • Não compartilhe o link do sistema
  • Se precisar dar acesso a funcionários, compartilhe só a planilha
  • Mude a senha do Gmail regularmente

Compartilhar com Funcionários

  1. Abra a planilha no Google Sheets
  2. Clique em "Compartilhar" (canto superior direito)
  3. Digite o email do funcionário
  4. Escolha: "Editor" (pode mexer) ou "Visualizador" (só ver)
  5. Não dê acesso ao Apps Script!

Restaurar Backup

Se precisar restaurar dados antigos:

  1. Vá no Google Drive
  2. Clique em "Novo" → "Upload de arquivo"
  3. Selecione seu arquivo de backup
  4. Copie os dados para a planilha atual

PASSO 11: Usando os Dados para Crescer

1Análise Mensal

Todo mês, após o fechamento, analise:

  • O valor do estoque cresceu comparado ao mês passado?
  • Quantos produtos novos você adicionou?
  • Quais categorias estão vendendo mais?

2Identificar Oportunidades

Use o histórico de movimentações para descobrir:

  • Produtos que vendem rápido: Compre mais, negocie melhor preço
  • Produtos que vendem devagar: Considere promoções ou não repor
  • Horários de maior movimento: Para estar sempre disponível

3Planejamento Financeiro

Com os dados do sistema você pode:

  • Calcular quanto lucro teve no mês
  • Saber quanto dinheiro está "parado" em estoque
  • Planejar compras com base no giro dos produtos
  • Negociar com fornecedores baseado no volume

Exemplo de Análise Mensal:

Janeiro: 150 produtos, R$ 5.000 em estoque

Fevereiro: 180 produtos, R$ 6.200 em estoque

Conclusão: Cresceu 20% em produtos e 24% em valor - negócio está indo bem!
